# Simple Chatroom

### Install

```sh

git clone https://github.com/colindev/chatroom.git $HOME/chatroom
cd $HOME/chatroom
./chatroom-server

```

then open `http://localhost:9000/client.html` on your browser

### Cli tool

```sh

./chatroom-cli -addr 'ws://localhost:9090' -name [your name] -room [room name]

```
